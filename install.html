1.安装与激活

安装与激活

1.安装环境要求

（1）必须安装在windows系统的收银台电脑，支持Windows XP、 Windows Vista、Windows 7、Windows 8、Windows 10等操作系统。

（2）收银台电脑必须安装 Microsoft .NET Framework 4.0（或以上版本）.NET服务。

（3）收银台电脑必须可以连接到网络（或通过代理服务器连接到网络）。

2.收银台电脑没有安装.NET 4.0 或没有连接外网的解决方案

2.1没有安装.NET的解决方案

注意：如果没有安装.NET4.0或以上版本，将无法在该电脑安装客户端。

（1）打开安装包，找到“独立安装.NET服务”并双击运行。

（2）点击“安装”以安装.NET服务。

（3）点击“完成”结束.NET安装。

2.2安装.NET失败的解决方案

安装.NET失败可能有以下几种原因导致：

2.2.1电脑上已经安装了其他版本的.NET。

这时可以打开控制面板，找到程序与功能，打开后在列表内查看.NET的版本。

如果.NET版本高于4.0，则可以直接安装客户端；如果没有看到.NET服务，则需要点击“打开或关闭Windows功能”。

在这里将低于.NET4.0的版本勾选项去掉，点击确定后重启电脑，并重新安装.NET。

2.2.2电脑之前安装过.NET，但是程序被删除，但注册表有残留。

（1）下载.NET清除工具来清除.NET残留文件。

工具下载链接：http://www.onlinedown.net/soft/27518.htm

（2）下载完成后，双击安装，点击“next”。

（3）选择第一个“I accept the License Agreement”后点击“next”。

（4）继续点击“next”。

（5）点击“完成”结束安装，找到Windows Installer Clean Up程序并打开，找到我们需要删除的程序。

选中后点击“Remove”即可。

2.2.3定制版收银机，电脑的注册表被锁定，无法修改和新增。

遇到这种情况只能重装系统。

2.3收银台没有连接外部网络的解决方案

如果收银台没有连接外网，客户端将无法被激活并且无法进行收银。因此我们必须将客户端安装在能够连接到网络的收银台电脑上。如果店内有一台电脑能连接到外络，我们可以在该电脑上安装代理服务器，使不能联网的电脑通过代理服务器实现联网。

（1）首先将完整的客户端安装包复制到连接到外网的电脑上，然后打开安装包，找到“辅助工具-代理服务”并双击打开。

（2）选择“简体中文”并确定。

（3）一直点击“下一步”。

（4）点击“完成”结束安装代理服务器，并在桌面上找到代理服务器的图标，双击并打开。

（5）保持代理服务器状态为运行中，点击“代理配置”。

（6）选择一个“本机监听的IP”。

（7）点击"确认"保存。

（8）然后在安装客户端完成之后，在客户端网络设置中，开启代理服务器功能。

（9）IP地址填写代理服务器监听的地址，端口填写代理服务器HTTP代理地址。

（10）最后保存后即可正常使用。

3.安装客户端

（1）首先要区分电脑系统是32位的还是64位的，如果无法判断，可以使用安装包内的工具“辅助工具-系统版本”来检测。

（2）双击后会弹出一个系统信息。

以上图为例，根据系统类型选项，确定该电脑是64位的操作系统。

（3）双击对应的安装包，点击”下一步“。

（4）选择”我接受“并点击”下一步“，一直持续到开始安装。

（5）点击”完成“结束安装。

4.激活客户端

（1）打开客户端，会弹出一个激活窗口。

（2）输入激活码后点击立即激活，提示激活成功后客户端就可以正常使用了。
